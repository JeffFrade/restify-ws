- URL do curso: https://www.youtube.com/watch?v=J3C63knfC3k&index=3&list=PLy5T05I_eQYO5Y3S3kVqBxQzkUNllPazF

Criação de um WebService RestFul com Node.js e Restify.

# Instalação e execução do projeto do curso:

- Instalação:
`npm install`

- Execução:
`npm run dev`

- Caminho:
`localhost:3000`
